# Filtro de Busca Dinâmico
Projeto desenvolvido simulando um filtro de busca dinâmico, onde os filtros buscam os valores dos imóveis independentemente ou em conjunto.
# Tecnologias
VueJs, JavaScript, Bootstrap, CSS
## Project setup
```
npm install
```
